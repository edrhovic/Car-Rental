About Us Header Image

The about-header.jpg file should be placed in this directory. It will be used as the header image for the About Us page.

Recommended specifications:
- Dimensions: 1920x500 pixels (or similar widescreen format)
- Content: An image representing Japanese cars, ideally showing a collection of JDM vehicles
- File size: Optimized for web (< 500 KB if possible)
- Format: JPG

If you don't have an appropriate image, you can use a stock photo or create a simple gradient background.
